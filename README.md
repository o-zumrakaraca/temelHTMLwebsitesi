# Çok Sayfalı HTML Web Sitesi Yapısı
Bu proje [Roadmap.sh HTML Projesi](https://roadmap.sh/projects/basic-html-website) referans alınarak yapılmıştır.

Bu proje, bir web sitesinin temel omurgasını ve anlamsal (semantic) HTML yapısını anlamak amacıyla geliştirilmiştir. Projede CSS kullanılmamış, tamamen HTML etiketlerinin doğru kullanımına odaklanılmıştır.

## 🎯 Projenin Hedefleri
- Birden fazla sayfadan oluşan bir web sitesi oluşturmak.
- `<header>`, `<nav>`, `<main>`, `<footer>` ve `<article>` gibi anlamsal HTML5 etiketlerini kavramak.
- Tüm sayfaları birbirine bağlayan ortak bir navigasyon menüsü kurmak.
- Sayfa başlıklarına (head) temel SEO meta etiketlerini eklemek.
- Kullanıcıdan veri almak için form (`<form>`, `<input>`, `<textarea>`) yapısını oluşturmak.

## 📂 Dosya Yapısı
- `index.html` (Ana Sayfa)
- `projeler.html` (Projeler listesi)
- `makaleler.html` (Blog/Makale yapısı)
- `iletisim.html` (İletişim formu)
- `egitim.html` (egitim bilgileri)

## 🚀 Sonraki Adımlar
Bu projenin tamamlanmasının ardından, oluşturulan bu sağlam HTML iskeletine CSS eklenerek tasarım (styling) aşamasına geçilecektir.
